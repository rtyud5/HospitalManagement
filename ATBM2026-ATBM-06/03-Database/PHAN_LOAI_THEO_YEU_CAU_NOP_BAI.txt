Ánh xạ nhanh script với yêu cầu đề (“Script CSDL + Data + Function & Stored Procedure”):

1) SCRIPT CSDL (DDL / schema / user / orchestration SYSDBA-ADMIN)
   • initDB.sql, run_01_as_sysdba.sql, dropAllTable.sql, dropAllUser.sql, fix_cleanup_sys.sql
   • Thư mục Admin/*.sql — PH1 quản trị (bootstrap, PKG_ADMIN…)
   • Thư mục OLS/*.sql — cấu hình/bảng/policy OLS cho THONG_BAO
   • RBAC/*.sql — tạo role, grant đối tượng Oracle
   • VPD/*.sql — package/function context + policy DBMS_RLS
   • Audit/*.sql — bảng audit, trigger/policy liên quan

2) SCRIPT DATA
   • insertData.sql (+ phần tạo user hàng loạt nếu cần báo cáo tách riêng: createUser.sql)

3) FUNCTION & STORED PROCEDURE
   • VPD/01_fn_get_role.sql — và các CREATE FUNCTION được gọi từ chính sách VPD (nếu có)
   • Admin/01_pkg_admin.sql — package PROCEDURE PKG_ADMIN và thân package (PH1)

Thứ tự chạy tổng thể xem README.md và run_02_as_admin.sql trong thư mục này.
